## 👋 Giới Thiệu

Chào mừng đến với GitHub của tôi! Tôi là **Vũ Tân Hưng** - kỹ sư Cơ điện tử chuyên sâu về **SLAM, Navigation và Autonomous Systems**. Với kinh nghiệm thực chiến trải dài từ **AMR công nghiệp** đến **Humanoid/Legged Robot**, tôi tập trung vào xây dựng hệ thống tự hành hoạt động tin cậy trong môi trường thực tế phức tạp.

### 🦾 Hiện Tại — VinDynamics (2026)

Đang phát triển hệ thống **SLAM & Navigation cho Humanoid Robot** tại VinDynamics:

- 🗺️ **3D SLAM**: LiDAR-based SLAM với GLIM (factor graph, GPU-accelerated)
- 🏔️ **Elevation Mapping**: GPU terrain mapping real-time cho locomotion (CUDA/CuPy)
- 🧭 **3D Navigation**: Full navigation stack (3D perception → global plan → MPC local planner)
- 🌿 **Terrain Analysis**: Ground segmentation (Patchwork++), traversability, terrain classification
- 📦 **Deployment**: Debian packaging, systemd service integration, CI/CD pipeline

### 🏭 Nền Tảng — AMR Công Nghiệp

Đã **thiết kế, phát triển và triển khai** thành công nhiều loại AMR cho môi trường công nghiệp:

- 🚗 **Differential Drive**: Robot di chuyển cơ bản với 2 bánh dẫn động độc lập
- 🔄 **Swerve**: Hệ thống di chuyển phức tạp với cụm bánh Steering-Traction
- ⚡ **Mecanum**: Robot đa hướng
- 🏗️ **Forklift AGV**: Robot nâng hạ Pallet tự động cho nhà xưởng, kho
